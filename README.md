---

## 🛠️ تکنولوژی‌های به کار رفته

- HTML5 / CSS3 (Flexbox، Grid، Media Queries)
- JavaScript (ES6)
- Clipboard API با Fallback `document.execCommand`
- localStorage برای ذخیره‌سازی سمت کاربر

---

## 📱 ریسپانسیو بودن

این ابزار برای تمام اندازه‌های صفحه بهینه شده است:

- روی دسکتاپ: فرم دوستونه با لیست دامنه در کنار
- روی موبایل: فرم تک‌ستونه، دکمه‌های بزرگ و قابل لمس

## 🧪 توسعه و مشارکت

اگر می‌خواهید تغییراتی ایجاد کنید یا باگ‌ها را گزارش دهید:

1. مخزن را **Fork** کنید
2. تغییرات خود را در شاخه `main` اعمال کنید
3. یک **Pull Request** ارسال کنید

برای اجرای محلی با قابلیت کپی (نیاز به HTTPS):

```bash
git clone https://github.com/worldof01/Netlify-V2ray-Generator.git
cd Netlify-V2ray-Generator
# سپس از افزونه Live Server در VS Code استفاده کنید
